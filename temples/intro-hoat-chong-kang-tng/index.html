法藏講堂（台南）

更新：五月 26, 佛曆2565 (2022)
GitHub
简体
字+ 字-

法藏講堂主供釋迦牟尼佛，其住持慶定法師，14歲在國內傳統佛教出家學法，後認同四念處是可見、可知實際的佛法後，開始推動原始佛法，提供法藏講堂作為原始佛教學院( Original Dhamma Institute )，供新雨社辦禪修及演講，也與其它道場共同推展原始佛法。（取材自：法藏講堂 （法藏講堂 — 文化資源地理資訊系統）
