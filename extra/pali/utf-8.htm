如何正常瀏覽以 UTF-8 編碼之網頁

　　因為有些巴利語或罕用漢字，以 萬國碼 Unicode 之 (UTF-8)" 表示，較為方便於全球通用閱讀。

請依照下列步驟：

  • 在 Google 之 Chrome 網頁瀏覽器下：
    1. 點選網址工具列最右邊第二個文件圖示
    2. (下拉視窗)找到 "編碼 (E)"
    3. 點選 萬國碼 Unicode (UTF-8)

  • 在 Mozilla 之 Firefox (火狐) 網頁瀏覽器下：
    1. 點選 "檢視 (V)"
    2. (下拉視窗)找到 "語言及字元編碼"
    3. 點選 萬國碼 Unicode (UTF-8)

  • 在 m$ 之 I.E. (微軟 Internet Explorer網際網路瀏覽器)下：
    1. 點選 "檢視 (V)"
    2. (下拉視窗)找到 "編碼 (D)"
    3. 移到 "其他" (M)
    4. 點選 Unicode (UTF-8)

  • 在 Apple 之 Safari 網頁瀏覽器下：
    1. Mac OS 下：
      1. 點選 "檢視 (View)"
      2. (下拉視窗)找到 "文字編碼 (Text Encoding)"
      3. 點選 Unicode (UTF-8)

    2. m$ Windows 下：
      1. 點選 "編輯 (Edit)"
      2. 點選 "偏好 (Preferences)"
      3. 點選 "外觀 (Appearences)"
      4. 點選 Unicode (UTF-8)
    (抱歉，很久沒用它了；細節翻譯自英文版操作程序 :-)

  • 在 Opera Software 之 Opera 網頁瀏覽器下：
    1. 點選 "檢視 (View)"
    2. (下拉視窗)找到 "編碼 (Encoding)"
    3. 點選 Unicode (UTF-8) (抱歉，很久沒用它了；細節翻譯自英文版操作程序 :-)
